🎮 کلاس برنامه‌نویسی کودکان

یادگیری با بازی و تمرین واقعی

شروع تمرین

✳️ عملگرهای رشته‌ای در پایتون

➕ اتصال (+)

first = "Ali"
last = "Ahmadi"
print(first + " " + last)

🔁 تکرار (*)

text = "Hi "
print(text * 3)

🔍 بررسی وجود (in)

print("python" in "python class")

🚫 عدم وجود (not in)

print("java" not in "python class")


✂️ اندیس و اسلایس در رشته‌ها



🔢 اندیس (Index)


text = "Python"
print(text[0])   # P
print(text[1])   # y




🔙 اندیس منفی


text = "Python"
print(text[-1])  # n
print(text[-2])  # o




✂️ اسلایس (Slice)


text = "Python"
print(text[0:3])  # Pyt




⚡ اسلایس با step


text = "Python"
print(text[0:6:2])  # Pto
